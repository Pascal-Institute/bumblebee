## 0. How to use? (Please Check Latest Version)

### build.gradle.kts
```kotlin
repositories {
    maven{
        url = uri("https://jitpack.io")
    }
}

dependencies {
    implementation ("com.github.volta2030:bumblebee:1.0.13")
}

```

### build.gradle

```groovy
repositories {
    maven {
        url 'https://jitpack.io'
    }
}

dependencies {
    implementation "com.github.volta2030:bumblebee:1.0.13"
}
```
